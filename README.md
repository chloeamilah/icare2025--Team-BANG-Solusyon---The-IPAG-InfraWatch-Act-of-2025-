# ICARE 2025 POLICY HACKATHON — OPEN DATA REPOSITORY  
### Theme: **Big Data, Bold Reforms: Transforming Big Data into Smart Policies for Better Governance**

This repository contains two official datasets curated for the ICARE 2025 Policy Hackathon. It supports participants in creating evidence-based policy proposals using real Philippine governance data.

---

##  Included Datasets

### **Dataset 1 — Flood Control Infrastructure Projects**  
Source: Sumbong sa Pangulo, DPWH  
Contains geocoded infrastructure project data: contracts, completion dates, durations, implementing offices, and contractor networks.

### **Dataset 2 — COMELEC × LGU Fiscal Panel Data**  
Combines COMELEC election results with BLGF fiscal performance: expenditures, revenue sources, IRA reliance, political competition metrics, and governance indicators.

## Purpose

This repository enables participants to:

- Identify corruption vulnerabilities using real datasets  
- Perform data-driven governance analysis  
- Develop actionable policy recommendations  
- Produce policy-ready insights aligned with the ICARE rubric

## Ethical and Responsible Use

Interpret results responsibly and avoid making allegations against specific persons.
